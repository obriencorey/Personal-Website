Electrical Stuff!




Controlling a 4 Digit 7 Segment Display


Since I did not have a bit shift register, I used discrete transistors to achieve something similar to multiplexing. Transistors were used to complete the individual cathodes of the display and allow for the appropriate digit to light up.

Additionally, I displayed the temperature using a weather api and experimented with some 7 segment animations to make things more exciting.


Learn More>>








Making a Turtle Robot


Inspired by Seymour Papert's book titled "Mindstorms," I set out to create an arduino based robot that would be able to input simple commands from the user like [line 10] and draw a 10cm line. I used a simple H bridge motor controller and plan to implement my own form of rotary encoding using light sensors.


Learn More>>








Displaying the Time With One 7 Segment Display


Much like the pseudo mulitplexing project, the origins of this projects come from frugality. I had recently received some internet capable microcontrollers and wished to write a simple program that got the time from the internet and displayed it. I only had one seven segment display, so I made an animation and displayed each digit one after the other.

Flying With Foamboard


Flitetest is a great organization that brings education and flying together in one of the best ways possible. I had to have one of their designs, and the Storch was the one I went with. This thing is insanely tough, surviving all of my beginner mistakes.

Building Balsa Airplanes


While foamboard is a great material and has incredible properties, sometimes it's nice to make planes the good old fashioned way: with wood. I am always trying new designs and aiming to find the perfect balance of performance and looks when it comes to design, and the plane to the left shows it. Instead of using coins or other forms of nose weight, I made a nose and my girlfriend painted it to resemble a P-40 nose. Of all the balsa planes I have ever made, this one both looked an performed the best.
