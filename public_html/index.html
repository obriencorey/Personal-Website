Testing Homemade Rocket Motors

This project consisted of making my own rocket motors, designing and building a stand capable of measuring their thrust, and finally calculating the specific impulse of each motor in order to enhance future rocket endeavors.


The CAD model of the rocket stand and the final product




A burned motor and the results of a full day of rocket science




Two different motor tests after change in nozzle geometry, resulting in specific impulse increase from 105s to 130s. Additionally, burntime was decreased resulting in more efficient motor



ESP8266 Based Messaging Service

The goal of this project was to create a messaging service using this website, an ESP8266 microcontroller, and an EInk display to display messages sent from the web. Additionally, this project included 3D modeling and printing a case, and creating an encrpytion algorithm written with PHP and a complementary decryption algorithm written with C++ for the ESP8266.


CAD Model and final product complete with status LED and button to dismiss new message




C++ program and assembling the final product




An example of the encrpyted messages. The PHP script also keeps track of who sent the message and when the message was sent. The same C++ program for the ESP8266 can decode this log of messages.
